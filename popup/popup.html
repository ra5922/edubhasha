🔑 Google Translate API Key
Adding a Google API key enables full-page translation of entire articles in one call. Without it, the free API translates in small chunks (may be slow for long pages).
How to get a free API key:
1. Go to console.cloud.google.com
2. Create project → Enable "Cloud Translation API"
3. APIs & Services → Credentials → Create API Key
4. Paste it above and click Save
